Steps to setup your scaper
1. install nodejs in your pc
2. you need to install these libraries first puppetter, fs, request, child_process and fast-csv.
3. Now make sure you have all the files in a single directory.
4. Now open the URL folder in their you will find a csv file called olx-url.csv.
5. Open it and put the file names and url like this (File Name, url) So that the numbers can save in the specific file name of that particular url. PS: their are already file names and url in it for demonstration.  
6. Now open the Command Line or Terminal and move to the directory where fork.js is present. for (e.g) cd c:\Users\user\node_modules\myDirectory and hit enter
7. Now just type node fork.js and sit back enjoy.
